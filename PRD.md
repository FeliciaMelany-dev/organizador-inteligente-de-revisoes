# 📚 Organizador de Estudos

## Problema
Pessoas que estudam programação têm dificuldade em organizar o que já estudaram e acompanhar termos importantes.

## Usuário
Estudante iniciante em programação que estuda sozinho.

## Solução
- Criar estudos (título + assunto)
- Marcar termos como aprendidos
- Visualizar estudos cadastrados

## Decisões Técnicas
- GET → buscar dados
- POST → criar estudos
- DELETE → remover
- PATCH → atualizar status de termos
- PUT → editar estudos
